阿b视频笔记常用的Linux命令介绍：13个基本命令和Shell脚本编程

windows系统装Linux系统：Windows Subsystem for Linux

13个基本命令

ls： 列出当前目录下所有文件
ls -l： 列出每个文件的属性 d开头说明是目录 -开头说明是文件 rwx是所有者的权限缩写
ll： ls -l的缩写
ls -a： 列出隐藏文件
ls -la： 列出隐藏文件的详细属性
cd 目录名：切换目录
cd ..： 进入上层目录
cd .： 进入当前目录
cd ../..： 进入上上层的目录
cd -： 返回上层
pwd： 打印当前路径
cat 文件名： 查看文件内容
head 文件名： 只看文件的开头
head –lines=2 文件名： 只看文件开头的两行
tail –lines=2 文件名： 只看文件尾部的两行
less 文件名： 看完整的文件，按Q退出阅读
more 文件名： 看完整的文件，按Q退出阅读

less和more的区别：

  1. less可以按键盘上下键显示上下文内容，而more不能通过上下方向键控制显示
  2. less不必读取整个文件，加载速度会比more更快
  3. less退出后shell不会留下刚显示的内容，而more退出后会显示刚显示的内容
  4. 阅读到文件结束时，less不会退出，而more会
  5. 相比more，less提供更加友好的检索，高亮显示等操作
  6. more会显示阅读进度，less不会

nano 文件名： 编辑文件
vim 文件名： 编辑文件，按I进入编辑模式，按Esc退出编辑模式，按:wq保存并退出，按：q!不保存强制退出
